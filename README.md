## Users テーブル

| Column | Type | Options |
|:----|:----|:----|

| email | string | null: false |
| password | string | null: false |
| family_name | string | null: false |

| first_name_kana | string | null: false |

### Association
- has_many :items
- has_many :purchasers

| Column | Type | Options |

## Purchasers テーブル
